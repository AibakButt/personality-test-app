
# Personality Test App

A personality test app, that can judge your personality of either Introvert or Extrovert. The application has a landing screen, a dynamic screen that reads questions and answers. And a finish screen that displays the personality trait.


# Getting Started
## Installation
 ```cd frontend && npm install```

```cd backend && npm install```

## Start the app
In frontend folder, run command
```npm start```

In backend folder, run command
```npm run start:dev```


## Run Tests
In frontend folder, run command
```npm run test```

In backend folder, run command
```npm run test```
